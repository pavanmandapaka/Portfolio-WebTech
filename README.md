# Pavan Sai's Portfolio

A modern, interactive portfolio website built with React, Vite, and GSAP featuring smooth animations, dark mode design, and responsive layouts.

##  Live Demo

- **Vercel:** https://portfolio-web-tech.vercel.app/



##  Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
